# Использование

1. Первый запуск:
```
# Подгрузка зависимостей во рабочую директорию (WORK_DIR="/var/lib/zapret-service")
sudo nix run github:pialtor/zapret-linux-flake --download-deps
```

2. Последующий запуск:
```
nix run github:pialtor/zapret-linux-flake
```
